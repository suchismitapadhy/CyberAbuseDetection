How Does This Work?

One of the major issues that has adversely affected social interactions online is the malice of cyber abuse. This abuse has taken many forms ranging from insults, threats of physical violence to propaganda-based smear campaigns and identity-based bullying. We have built a machine learning model using Natural Language Processing capable to detecting personal insults directed at a person through machine learning of abusives. We then use it to predict whether text a person intends to post online is possibly abusive in native. This can server to warn users so that they can take corrective action before posting the text.

About Us

We are a team of college students working on freeing the web of abusive content. A first step in this direction is to warn users before they post abusive content so that they can take corrective action. Another use of this system is to flag users who frequently post offensive content online.
